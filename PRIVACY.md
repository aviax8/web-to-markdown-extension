# Privacy Policy for Web to Markdown

Last updated: February 2026

## Summary

Web to Markdown does not collect, transmit, or sell personal data.

All conversion is done locally in the browser.

## Data collection

The extension does not:

- collect personal information
- track browsing history
- upload page or chat content to external servers
- use analytics or telemetry

## Data processing

When you run a context menu action, the extension:

1. reads content from the active page
2. converts HTML to Markdown locally
3. writes markdown to your clipboard

No content is sent to the developer.

## Stored data

The extension stores only user settings in browser extension storage:

- AI request header text
- AI request quote prefix
- AI response header text

These settings may sync across your own browser profile devices if browser sync is enabled.

## Permissions

- `clipboardWrite` - copy markdown to clipboard
- `storage` - save settings
- `contextMenus` - provide right-click menu actions

## Contact

For questions, open an issue on the project repository.
